Introduction

Drizzle ORM is a lightweight TypeScript ORM that feels like writing SQL — with full type safety and zero runtime overhead.

import { drizzle } from 'drizzle-orm/postgres-js' import postgres from 'postgres' const db = drizzle(postgres(process.env.DATABASE_URL))

Why Drizzle?

Unlike other ORMs, Drizzle gives you a SQL-like query builder that maps 1:1 to the queries being executed. No magic, no hidden N+1 issues — just transparent, predictable database access.

Full TypeScript inference means your editor knows the shape of every query result without any additional type annotations.
